# 本地开发

## 安装依赖

根据官方文档，项目需要：
- 需要node >=22 <25。

部分脚本依赖 Bun（现代 JavaScript 运行时）

```bash
# 安装 npm 依赖
npm install

# 如果使用 Bun（推荐，更快）
bun install
```



## 启动方式

### 无窗口 WebUI 模式（推荐，轻量）

这种模式不启动 Electron 桌面窗口，只启动 WebUI 服务器，通过浏览器访问。

```bash
bun run webui
```

这是从源码直接运行 WebUI 的最简方式，适合：
- 只需要远程访问
- 不想启动完整的桌面应用
- 在服务器上部署



### 远程访问 WebUI 模式
如果需要从其他设备（手机、另一台电脑）访问：

```bash
bun run webui:remote
```
这个命令会：

绑定到 0.0.0.0（允许外部访问）

自动显示局域网访问地址

### 纯服务器模式（无 Electron 依赖）

适合在 Linux 服务器上长期运行：

```bash
bun run server:start
```

特点：
- 无 Electron 依赖，体积更小
- 适合树莓派、NAS、云服务器等环境
- ⚠️ 部分 Electron-only 的 Bridge 功能在此模式下不可用
